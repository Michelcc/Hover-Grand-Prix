# Hover Grand Prix


Juego de carreras arcade con hovercrafts en Unity 3D. Controlas a **Riko** en un circuito rápido y peligroso, compitiendo contra el rival IA **Vortex**.  

## Mecánicas principales
- Manejo arcade: aceleración, freno, giro y drift ligero  
- Boost: barra de velocidad temporal  
- Checkpoints y respawn  

## Requisitos
- Unity 2022.2.9f1  
- Assets: vehículos, circuito, boost pads, partículas y sonido  

## Objetivo
Completar la vuelta más rápido que **Vortex** en un circuito lleno de curvas, rampas y atajos.
